# qa-full

Este workflow ejecuta el proceso completo de QA sobre una Historia de Usuario.

Incluye:
- análisis de HU
- validación de código
- validación de base de datos
- trazabilidad
- evaluación por reglas
- medición de cobertura
- sugerencias de mejora
- generación de pruebas

Es el flujo principal del sistema QA en su versión completa.
